How to Use This Tool

  • 1. Enter ZIP codes in the text area above (either comma-separated or one per line)
  • 2. Set your desired search radius (0-100 miles)
  • 3. Click "Show on Map" to visualize your selected ZIP codes in blue
  • 4. If radius > 0, surrounding ZIP codes within the radius will appear in gray
  • 5. Hover over any ZIP code to see its number
  • 6. Click on any gray ZIP code to add it to your selection
Tips:
  • • Use the fullscreen button in the top-left corner for a larger view
  • • You can zoom and pan the map freely to explore different areas
  • • New ZIP codes will be added in the same format you initially used (commas or newlines)
